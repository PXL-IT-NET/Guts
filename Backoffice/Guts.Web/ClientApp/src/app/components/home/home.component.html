GUTS

With GUTS (Growing towards a Unit Testing strategy) you can make programming exercises and immediately get feedback on your progress by running the automated tests.
Test results are sent to the GUTS api. In a later fase you will be able to review your results online.

Supported courses:

  • .NET Essentials. Download exercises via github and send test results via Visual Studio
  • .NET Advanced. Download exercises via github and send test results via Visual Studio
